cuckoo-tools
============

Tools and configs for my Kali installation.

Setup
=====

I tested this script on Debian 8.3

    sudo apt-get install git
    git clone 

Start Cuckoo:

    cd ~/src/cuckoo
    ./cuckoo.py

Start Cuckoo web interface at http://127.0.0.1:8000/:

    cd ~/src/cuckoo/web
    python manage.py runserver
